# 📄 Resume Scoring using NLP

An AI-powered Resume Scoring System that analyzes resumes using Natural Language Processing (NLP) techniques and generates a score based on the quality of the resume, extracted skills, and relevance to job requirements. The project helps job seekers evaluate and improve their resumes before applying for jobs.

---

## 🚀 Features

- 📂 Upload resume (PDF/DOCX)
- 📝 Extract text from resumes
- 🔍 NLP-based text preprocessing
- 💡 Skill extraction
- 📊 Resume scoring
- 🎯 ATS-friendly evaluation
- 📈 Personalized improvement suggestions
- ⚡ Fast and automated analysis

## 📌 How It Works

1. Upload a resume.
2. Extract text from the document.
3. Clean and preprocess the text using NLP.
4. Identify important keywords and skills.
5. Calculate the resume score.
6. Display suggestions for improvement.
